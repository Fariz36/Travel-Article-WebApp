## ⚙️ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Fariz36/Travel-Article-WebApp
cd my-app
```

### 2. Install the dependencies
```bash
npm install
```

### 3. Run the development
```bash
npm run dev
```

Open http://localhost:3000 to see the app
